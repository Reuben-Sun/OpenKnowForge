# 代码示例2

## 场景

使用madmom库提取音频节奏点。

## 代码

```python
from madmom.features.beats import RNNBeatProcessor, CRFBeatDetectionProcessor
from madmom.features.downbeats import RNNBarProcessor

def get_beats(input_audio, threshold_value=0.37):
    print(f'Process {input_audio}')
    proc = CRFBeatDetectionProcessor(fps=100)
    act = RNNBeatProcessor()(input_audio)
    data = proc(act)	# 非常密的节奏点

    proc2 = RNNBarProcessor()
    downbeat_prob = proc2((input_audio, data))
    ans = np.around(downbeat_prob, decimals=3)	
    filtered = arr[arr[:, 1] > threshold_value]		
    first_column = filtered[:, 0]	# 根据强度筛选的重音

    return json.dumps(ans.tolist()), json.dumps(data.tolist())
```

## 说明

- `RNNBeatProcessor` + `CRFBeatDetectionProcessor` 用于提取较密集的节奏点。
- `RNNBarProcessor` 可进一步给出下拍概率。
- `threshold_value` 可用于筛选较强的重音时间点。
